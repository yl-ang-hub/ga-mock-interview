# Mock Challenge

Create a shoe-tracking system to hold, at minimum, the following data within the inventory:
- Type (e.g., track shoes, high heels, etc.)
- Brand
- Size
- Colour
- Miscellaneous

Requirements:
1. Develop RESTful APIs using any framework and programming language of your choice.
2. Use an SQL database for data storage.
3. Implement full CRUD functionality.
